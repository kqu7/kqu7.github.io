Kaggle Cassava Leaf Disease Image Classification Task (Spring 2021)
Individual Project
We implemented four different algorithms (branch and bound, approximation, and two variations of local search algorithms) attempting to cope with a well-known NP-hard problem in graph theory, the minimum vertex cover (MVC). We measured the performance of different algorithms in terms of running time and relative error, finding that one variation of the implemented local searches performed best in terms of the above two criterias but the performance depended significantly on the starting point of the search.
In addition, we analyzed the theoretical time and space complexities of the four algorithms, comparing the theoretical time complexities to the actual running time and providing reasons for some inconsistencies on certain graph instances.
The project report is available here. Full implementation is also available upon query.

Optimal Review Ranking for Improving Shopper’s Decision Making (Spring 2020)
Group members: Akhil Sai Peddireddy (ap3ub), Kai Qu (kq4ff), Rakshita Kaulgud Ramesh (rrk7pb), and Haoran Zhu (hz3fr) [All members were CS students at UVA.]
We developed a system that provides personalized ranking of customer reviews for each shooper. Using cumulative position-weighted score, we were able to demonstrate that there is an average of 20% increase in user satisfaction using our personalized ranking system compared to the default Amazon ranking system.
Also, we proposed a method of rating products at term level based on users’ preferences. For example, if a user cares very much about the price of a product, we’ll offer another rating of the product solely based on the price factor. This method can be applied to recommendation systems to provide accurate recommendations.
The project report is available here. Full implementation is also available upon query.
